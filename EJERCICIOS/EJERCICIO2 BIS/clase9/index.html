Variables CSS

La función var() se usa para poner una variable css y poder reutilizarla en nuestra hoja de estilo, sin necesidad de poner de manera repetitiva lo mismo.

Una buena manera de usar las variables CSS es cuando se trata de los colores de tu diseño. En lugar de copiar y pegar los mismos colores una y otra vez, puede colocarlos en variables.

Sintaxis

El nombre de la variable debe comenzar con (2 guiones medios) --nombre y distingue entre mayúsculas y minúsculas.

Ventajas

  • Hace que el código sea más fácil de leer (más comprensible).
  • Hace que sea mucho más fácil cambiar los valores de color.
  • Para cambiar el color azul y blanco a un azul y blanco más suave, solo necesita cambiar los dos valores de las variables.

Medias Query CSS

Las medias queries son un módulo de css, que en el desarrollo web nos permite adaptar el contenido de nuestra web a las características del dispositivo en el cual se abrió la página web. Más precisamente en lo que refiere a la resolución de la pantalla (tamaño).

Son una sintaxis de CSS que nos permite definir estilos que solo se aplican cuando dicha condición se cumple. Las medias query actúan como contenedores de estilos css.

Diseño Responsive

La forma de trabajo es usar un mismo diseño de página para que se adapte a todas las resoluciones. Gracias a las medias queries podemos implementar el diseño responsive, ya que se muestran o no las propiedades css, dependiendo de ciertas condiciones.

Condiciones / Resultados

  • Si la pantalla es pequeña.
  • Si la pantalla es grande.
  • Si está en posición horizontal.
